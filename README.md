# 📄 Professional Resume Generator

An interactive, responsive, and ATS-friendly **Resume Builder** web application built using **HTML5, CSS3, and JavaScript**. It features live real-time previewing, dynamic field additions, auto-hiding empty sections, and one-click PDF downloads!

---

## ✨ Features

- ⚡ **Real-Time Live Preview:** Watch your resume format instantly as you type your details.
- 🎨 **Multiple Template Options:** Switch seamlessly between Modern, Classic, and Minimalist templates.
- 🧠 **Smart Dynamic Hiding:** Optional fields automatically hide from the preview/PDF if left empty.
- ➕ **Dynamic Section Addition:** Easily add multiple entries for Education, Projects, Certifications, and Achievements.
- 💾 **Local Storage Integration:** Saves form state automatically in the browser.
- 📥 **Instant PDF Export:** Export crisp A4-formatted PDF resumes powered by `html2pdf.js`.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla ES6+)
- **Libraries:** html2pdf.js
- **Data Persistence:** Browser LocalStorage API
